ARCHITECTURE COMPONENTS DEFINED
================================================================================

POWERSHELL MODULE:
  - Module Name: FlashDB
  - Version: 0.1.0
  - Provider Interface: FlashdbProvider (abstract base class)
  - Enumerations:
    * FlashdbCreationMethod (BackupRestore, ReplicaBackup, TableByTableCopy)
    * FlashdbCloneStatus (Created, Attached, Detached, Corrupted, Expired)
    * FlashdbCheckpointPhase (PreETL, PostETL, Manual, Custom)

  - Core Functions:
    * VHDX: Mount, Unmount, Create Snapshot, Revert Snapshot
    * Metadata: Create, Save, Load, Log Operations
    * Provider: Register, Get (provider registry)

  - Public Cmdlets (30+):
    * Golden Image: New, Get, Update, Remove
    * Clone: New, Get, Connect, Disconnect, Remove
    * Checkpoint: New, Get, Set, Restore, Diff, Remove
    * Utilities: Test, Health Check, Storage Report, Config

SQL SERVER PROVIDER:
  - Class: SqlServerProvider : FlashdbProvider
  - Golden Image Creation Methods:
    1. BackupRestore - From backup file
    2. ReplicaBackup - From read-only replica (BACKUP FROM MIRROR)
    3. TableByTableCopy - Direct table copy
  - Database Operations:
    * Attach database to instance
    * Detach database from instance
    * Close active connections (force-close support)
    * Validate connections
    * Get database info (size, row counts, schema)
  - Helper Methods:
    * Compute row count hash
    * Detect replica lag
    * Enumerate tables

REST API:
  - Base URL: https://localhost:6060/api/v1
  - 30+ HTTP Endpoints across 4 categories
  - Golden Image Mgmt (6): Create, List, Get, Update, Refresh, Delete
  - Clone Mgmt (6): Create, List, Get, Attach, Detach, Delete
  - Checkpoint Ops (8): Create, List, Get, Update, Restore, Diff, Delete, Restore-Golden
  - Admin (5): Health, Storage, Config Get/Patch, Operations
  - Authentication: Windows integrated, JWT, API key
  - Error Handling: Standard HTTP status codes with JSON detail

GUI CLIENT:
  - Framework: WPF or Web-based
  - Views: Dashboard, Clone Management, Checkpoint Timeline, Diff Viewer, Settings
  - Features: Create clones, manage checkpoints, compare states, view storage
